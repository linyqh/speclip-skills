# Conflict Case

## Goal
测试 skill 是否会优先保留“婆家羞辱 / 护子 / 掀桌”这些最有戏的原片，而不是只保留信息说明。

## Inputs
- transcript 里包含：不给饭吃、逼认后爹、介绍改嫁、掀桌、争抢财物
- characters.md 已确认：蔡小燕、孩子、嫂子 / 亲戚
- draft_script.md 已写顺整段家庭压迫线

## What good output looks like
- 至少一个 `original` 段直接保留羞辱性原台词
- 至少一个 `original` 或强承载 VO 保留掀桌前后的情绪爆点
- VO 不把“她很惨、家里很坏”说两遍
