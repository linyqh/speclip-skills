# short-drama-video 技能栈分工

这是 `short-drama-video/` 目录下 3 个核心 skill 的协作说明。

## 推荐顺序

1. `short-drama-commentary`
   - 负责：结构、节奏、原片/解说穿插策略
2. `drama-script-writer`
   - 负责：把结构方案写成真正可用的 VO / 混排文案
3. `drama-explainer`
   - 负责：把结构和文案落成 `storyboard.json`、音频、匹配、成片

## 共同规则

3 个 skill 都必须遵守：

- `references/excellent-short-drama-commentary-playbook.md`

如果单个 skill 的局部说明与这份共享手册冲突，优先以共享手册为准。
