# short-drama-commentary

一个专门解决“短剧解说结构怎么搭、原片应该插在哪里、解说该带过哪里”的字幕优先型 skill。

它特别适合两类工作：

1. 拆解优秀短剧解说字幕，学习它们的结构和节奏
2. 根据原剧字幕 / 梗概，直接产出短剧解说的插片计划与混排脚本

它和现有 skills 的关系：

- `drama-script-writer`：偏写文案
- `drama-explainer`：偏完整成片工作流
- `short-drama-commentary`：偏结构设计、原片/解说穿插决策

## 默认交付

- 结构诊断
- 原片 / 解说插入规则
- 分段 beat sheet
- 可执行的混排脚本或插片计划表
